# 📝 Todo App

A simple and modern Todo App built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## ✨ Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 📊 Display total tasks
- 🌙 Toggle between Dark Mode and Light Mode
- 🎨 Clean and responsive UI
- ⚡ Built with pure JavaScript (No frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
